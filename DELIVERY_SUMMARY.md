   ↓

   ↓

### User Journey
1. User logs in and goes to dashboard
2. Sees Feature 3: "Skill Gap Analysis" with interactive button
3. Clicks button → navigates to `/skill-gap-analysis/`
4. Configures target role (job title, career path, industry, experience level, location)
5. Clicks "Run Market Analysis"
6. Views:
   - Market benchmark requirements
   - Visual comparison (radar & bar charts)
   - Skill gap analysis (met, missing, proficiency, emerging)
   - Priority matrix (critical to low)
   - Learning path with resources
   - Role comparison module
   - Progress tracker with notes
7. Tracks progress with checkboxes
8. Adds personal notes
9. Clicks "Dashboard" button to return
